---
name: pr-update
description: Update the description of an existing Pull Request (PR)
---

can you please review the main description for the current branch's PR?

do NOT make any edits in these sections
- summary
- linear ticket
- slack discussion links
- screenshots
- demos
- run command or verification instructions

for any other sections with bullet points, please read them and update or remove as necessary so that the PR description accurately summarizes the changes under each section heading.
